Score a task for ambiguity and iteratively refine it through clarifying questions.

**Usage:**

```
/maetdol:gate <task description>
```

Runs up to 3 rounds of ambiguity scoring with socratic clarification between rounds. Outputs refined requirements when the gate passes.
